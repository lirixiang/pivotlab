核心数据流
① 用户提问: Web UI → WebSocket → FastAPI → ReAct Loop
② LLM 推理: ReAct Loop → LLM Factory → Qwen/DeepSeek/SiliconFlow → 返回 tool_calls 或 text
③ 工具执行: ReAct Loop → 权限检查 → Tool Registry → 执行工具 → SQL Guard → 查询 PG
④ 结果返回: 工具结果 → ReAct Loop → LLM 综合分析 → WebSocket 流式推送 → 用户看到答案
⑤ 知识增强: kb_search → pgvector 向量相似度 → 注入上下文 → LLM 生成更准确的回答
